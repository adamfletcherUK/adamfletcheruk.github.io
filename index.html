    • 2019 - 2020

    Data Analyst

    Equal Experts
    Creating needs-based personas for a job-listing service:
  • The client wanted to better understand how users interacted with a job-listing website, with the aim of personalising user experiences based upon a set of needs they might have.
  • Working with a UX company we conducted interviews and designed surveys to gather information about what needs users were experiencing when applying for a job (roughly 100,000 respondents).
  • The dataset was incomplete to avoid asking respondents too many questions, so Boolean matrix-factorisation was used to predict user responses for unasked questions, followed by clustering to group users with similar needs.
  • The clusters were converted into personas with user stories and the business was now better able to respond to the user’s needs, improving user retention and paid user numbers.
  • Read the case-study writeup
    • Reducing the cost of variation in providing a service:

  • Working with a new workflow tracking system, we were able to attribute normal operating criteria and areas which were operating outside of them.
  • In a detailed report outlining standard operating procedures I also determined the cost-saving resulting from implementing the new procedures, which exceeded £2million.
    • Migrating a legacy data warehouse to modern infrastructure:
  • The client wanted to migrate a legacy SQL server 2000 data warehouse to a cloud solution.

  • I developed proof of concept ideas for a new architecture using data warehouses and marts to protect data security and integrity, looking to replace the aging ETL processes.
  • The client was able to determine the project direction and better understand the risks in delivering the project.

    • 2009 - 2011

    Research Scientist

    QIAGEN
    Reducing false-positive rate in cancer mutation detection by introducing known errors:
  • The project aim was to improve the stringency of a diagnostic test to determine chemotherapy options by purposely introducing errors into the product chemistry.
